About This App

This is a SPA - Single Page Applciation., where a SPA is a javascript based client side executing UX/UI framework powered Applciation having only one html page. All events and data operations and UI rendering are handled by the underling javascript code assissted by the asynchrnous rest-pai communication when needed for data related operations.

The current application is a HR management simulation tool that manages details of departments and its employees on a hypothetical data model.
